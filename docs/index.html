Drone Icarus · Developer Handoff v1.0

Everything engineering needs to ship Icarus.

A map-first marketplace for aerial drone footage. This handoff bundles product scope, proposed architecture, data model, API contract, design tokens, and a 26-week roadmap. The working clickable prototype ships alongside as a single HTML page — treat it as UX truth, not production code.
